

### $\gamma-\gamma$ 二重符合分析程序 Gate2

- 设ROOT文件名为 file.root, 二维 𝛾−𝛾 矩阵的名称为hg2 (TH2* hg2).
- 在终端命令行运行
```cpp
root -l 'gate2.C("file.root","hg2")'
```
进入ROOT后在ROOT命令行运行
```cpp
.x gate2.C("file.root","hg2")
```
